# Multimodal_Image_Capturing

# 🧠 Multimodal AI: Image Captioning System  
**Project 5 – Flickr8k Dataset**

## 📌 Overview
This project builds a Multimodal AI system for Image Captioning using the Flickr8k dataset.  
It combines computer vision (images) and natural language processing (text) to generate captions.

## 🎯 Objectives
- Build an image captioning model  
- Combine image + text data  
- Train deep learning models  
- Evaluate performance  

## 📂 Dataset
- Flickr8k Dataset (Kaggle)  
- 8,000 images with 5 captions each  

## 🧩 Features
- Image preprocessing  
- Text tokenization  
- CNN feature extraction  
- Caption generation model  

## ⚙️ Technologies
Python, NumPy, Pandas, TensorFlow/PyTorch, NLTK, Matplotlib

## 🚀 How to Run
```bash
pip install kaggle nltk torch torchvision transformers
```

Run the notebook in Jupyter or Colab.
